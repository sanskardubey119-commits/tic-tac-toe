# Tic-Tac-Toe AI with Alpha-Beta Pruning

A modern Tic-Tac-Toe game built using **Python** and **Tkinter**, featuring an intelligent AI opponent powered by the **Minimax algorithm with Alpha-Beta Pruning**. The game offers multiple difficulty levels, a clean graphical user interface, responsive gameplay, and a built-in scoreboard for an engaging user experience.

## ✨ Features

* 🎮 Interactive GUI built with Tkinter
* 🤖 Smart AI using Minimax with Alpha-Beta Pruning
* 🟢 Three difficulty levels:

  * Easy (Random Moves)
  * Medium (Rule-Based AI)
  * Hard (Optimal AI with Alpha-Beta Pruning)
* 📊 Live scoreboard (Player Wins, AI Wins, Draws)
* 🔄 Play Again and Reset Score options
* 📱 Responsive square game board
* ✅ Automatic win, loss, and draw detection
* ⚡ Fast AI decision-making with optimized search

## 🛠️ Technologies Used

* Python 3
* Tkinter
* Minimax Algorithm
* Alpha-Beta Pruning
* Object-Oriented Programming (OOP)

## 🧠 AI Algorithm

The Hard difficulty uses the **Minimax algorithm** enhanced with **Alpha-Beta Pruning**, allowing the AI to evaluate possible game states efficiently while eliminating unnecessary branches. This optimization enables the AI to make optimal moves and remain unbeatable.

## 📂 Project Structure

* GUI-based desktop application
* Object-oriented design
* Modular game logic
* Efficient AI search algorithm
